# Nick Tsiflikiotis

Founder. I design and ship agentic AI systems that run real operations — financial analysis, regulatory compliance, cross-border trade, and HR — then put them in front of real users.

### What I've shipped

**[Tsifulator.ai](https://github.com/Tsifulator/tsifulator.ai)** — An agentic workspace for financial analysts: one AI operating layer that works *inside* the tools analysts already live in, through five native add-ins — [Excel](https://github.com/Tsifulator/tsifulator-excel-addin), [Word](https://github.com/Tsifulator/tsifulator-word-addin), [PowerPoint](https://github.com/Tsifulator/tsifulator-powerpoint-addin), [Google Workspace](https://github.com/Tsifulator/tsifulator-google-addin), and [RStudio](https://github.com/Tsifulator/tsifulator-r-addin). The analyst asks once; the agent executes wherever the work lives.
<sub>Python · Office.js · Apps Script · R/Shiny · Claude API</sub>

**[ALTER EGO Intake](https://github.com/Tsifulator/alteregosoftware)** — Production candidate-intake system for a facility-management company. Walk-in applicants self-serve on a kiosk in any of **15 languages (RTL-aware)** and the record lands in the Workable ATS pre-filled — with a never-lose-a-candidate retry queue that survives network drops. Replaced the paper-form → manual-retyping loop entirely.
<sub>FastAPI · Railway · kiosk deployment</sub>

**[D&D Defense](https://github.com/Tsifulator/dd-defense)** — Audit engine for ocean-carrier demurrage & detention invoices under **FMC 46 CFR Part 541**. Two-layer design: *facial* audits prove violations from the invoice alone (missing required elements or late issuance can eliminate the obligation to pay); *substantive* audits check closures, container availability, and tariff rates against supplied evidence — and it drafts the dispute letter. Core pipeline is stdlib-only Python: no install, no API key.
<sub>Python · rule-engine design · regulatory NLP</sub>

**[Frozen Fish](https://github.com/Tsifulator/frozen-fish)** — A Boston → Tokyo premium-seafood brokerage where **the product is the business itself**: three Claude-powered agents (sourcing, matching, outreach) plus a deal database and daily digest let one operator compete with an industry still run on phone and fax.
<sub>Python · Claude agents · Airtable · bilingual (EN/JA)</sub>

**[ALTER EGO Scraper](https://github.com/Tsifulator/alteregoscraper)** — Autonomous lead-generation agent for integrated-facility-management prospects: sources, qualifies, and enriches leads end-to-end.
<sub>Python</sub>

**[Brands Box](https://github.com/Tsifulator/brands-box)** — Fast, minimal showroom site for an Athens fashion wholesaler.
<sub>Astro</sub>
